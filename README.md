# Customer Shopping Behavior Analysis

## Live Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, MySQL, Power BI, and Gamma. The goal is to analyze customer shopping behavior, uncover business insights, and present findings through interactive dashboards, reports, and presentations.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer demographics
* Product categories
* Purchase amounts
* Payment methods
* Purchase frequency
* Review ratings
* Shopping preferences

The data was used to identify customer trends, product performance, and purchasing patterns.

## Project Steps

### 1. Data Loading

### 4. SQL Analysis

### 5. Power BI Dashboard

### 6. Reporting & Presentation

## How to Run

### Clone the Repository

```bash
git clone <repository-url>
cd Customer-Shopping-Behavior-Analysis
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy pymysql
```

### Run the Jupyter Notebook

```bash
jupyter notebook
```

Open the notebook and execute all cells to perform data cleaning, EDA, and database operations.

### MySQL Setup

1. Create a MySQL database.
2. Update database credentials in the notebook.
3. Load the cleaned dataset into MySQL.
4. Run the SQL queries provided in the project.

### Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop.
* Refresh the data source if required.

### Presentation

* Open the Gamma presentation link or exported presentation file to review project findings.
